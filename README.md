## Settings

Configure in VS Code Settings:

```json
{
  "codesnap.theme": "dark",           // dark, light, monokai, nord, dracula
  "codesnap.showLineNumbers": true,   // Show/hide line numbers
  "codesnap.windowControls": true,    // Show macOS window controls
  "codesnap.shadow": true             // Add shadow effect
}
```
